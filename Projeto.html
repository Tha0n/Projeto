Um pouco da Historia

Hypertext Markup Language (HTML) foi criada por Tim Berners-Lee em meados de 1990. O objetivo era criar uma maneira de compartilhar documentos e pesquisas entre cientistas de diferentes universidades. A primeira versão foi publicada em 1991, consistindo em 18 tags. Desde então, cada versão do HTML vem com novas tags e atributos.

A partir da criação do W3C (World Wide Web Consortium), HTML5 vem sendo o novo padrão para estrutura e apresentação do conteúdo, sendo suportado por todos os navegadores.

O que são tags HTML

Todas as marcações são realizadas pelas tags que em geral são estruturas de linguagem de marcação que contém instruções. Os navegadores interpretam essas instruções e a partir disso renderizam uma pagina. As tags são responsáveis por criar elementos nas paginas web com a função de organizar, identificar e limitar o conteúdo de uma pagina. Cabe então ao Browser reconhecer esses elementos, e renderizá-las apresentando seu conteúdo aos usuários.

Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada. Uma tag HTML deve conter:

  1. Uma tag de abertura — começando com um símbolo < >;
  2. Conteúdo — instruções curtas para exibir o elemento na página;
  3. Uma tag de fechamento — terminando com um símbolo .

Tags mais comuns

Existem vários tipos de tags HTML. Cada uma tem um fim específico. Algumas das principais tags do HTML incluem:

  • < html > - usada para definir o início do documento HTML.
  • < head > - usada para incluir informações, como o título da página, meta tags e scripts.
  • < tittle > - usada para definir o título da página, que aparece na barra de título do navegador.
  • < body > - usada para definir o conteúdo da página, como textos, imagens, vídeos, links e outros elementos.
  • < h1 > - usadas para criar títulos e subtítulos, com diferentes níveis de importância.
  • < p > - tag usada para criar parágrafos de texto.
  • < a > - usada para criar links para outras páginas da web ou para arquivos no próprio site.
  • < img > - usada para inserir imagens na página.
  • < ul > e < li > - criam listas com marcadores.
